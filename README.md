## Running the code 

* Create a .venv and activate it using `source .venv/bin/activate` 
* install the modules needed using ` pip install pandas scikit-learn torch` 
* run the program from the root directory using `python nsl_kdd_fgsm.py` 
* for higher or lower attack strenghts, specify the epsilon value. Ex: `eps = 0.075`
